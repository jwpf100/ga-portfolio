Traffic Lights

A code challenge from General Assembly to fix a broken set of traffic lights.
Main Language: Javascript
Other Language: HTML; CSS;

Javascript Code:

            
            document.getElementById('stopButton').onclick = illuminateRed;
            document.getElementById('slowButton').onclick = illuminateYellow;
            document.getElementById('goButton').onclick = illuminateGreen;
          
            function illuminateRed() {
            clearLights();
            document.getElementById('stopLight').style.backgroundColor = "red";
            }
     
            function illuminateYellow() {
            clearLights();
            document.getElementById('slowLight').style.backgroundColor = "yellow";
            }
      
            function illuminateGreen() {
            clearLights();
            document.getElementById('goLight').style.backgroundColor = "greenyellow";
            }
    
            function clearLights() {
            document.getElementById('stopLight').style.backgroundColor = "black";
            document.getElementById('slowLight').style.backgroundColor = "black";
            document.getElementById('goLight').style.backgroundColor = "black";
            }

Essential Coaching

www.essentialcoaching.co.uk


This is my main 'side' project. A website for a career coach, showcasing her services, contact information, and blog.


The site is live, utilising bootstrap as a framework, hosted on Heroku, running on a node/express server, with js and CSS bundled with webpack.


The current site is stage one. The next stage will involve adding the following:

  • A functioning blog, utilising a database to store articles
  • Password access for site admin
  • A contact form
